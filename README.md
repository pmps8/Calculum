# Calculum

**Autor:** Pablo Peñaloza Sulbaran  
**Proyecto:** Calculum: Tu Calculadora de Magnitudes Físicas de Confiazna

 Descripción del Proyecto

Calculum es una aplicación web diseñada para calcular magnitudes físicas fundamentales de manera rápida y precisa.  
La herramienta permite obtener:

- Velocidad  
- Aceleración  
- Fuerza  
- Trabajo  
- Energía Cinética  
- Potencia  

El objetivo principal es servir como apoyo educativo para estudiantes que requieren resolver cálculos básicos de física de forma intuitiva.
